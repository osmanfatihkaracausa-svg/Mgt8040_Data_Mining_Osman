# Cost of Habits Tutorial

## Overview

This project was completed as a practice exercise during **MGT 8040 – Data Mining & Business Analytics** at **Georgia State University** under **Dr. Lee Frank**.

The purpose of this tutorial was to learn data preparation, data integration, exploratory analysis, and basic machine learning workflows using Python.

## Objective

The project combined two datasets:

* Personal Habits
* Daily Spending

The goal was to investigate how daily habits may relate to personal outcomes such as focus, spending, and overall performance.

## Skills Practiced

* Data Loading with Pandas
* Data Cleaning
* Date Formatting
* Left Joins
* Missing Value Handling
* Correlation Analysis
* Feature Engineering
* Exploratory Data Analysis (EDA)

## Dataset

The exercise used two sample datasets:

1. Personal Habits Dataset

   * Sleep Hours
   * Coffee Consumption
   * Work Focus
   * Exercise Metrics

2. Daily Spending Dataset

   * Spending Amount
   * Daily Purchases
   * Financial Activity

The datasets were merged using the Date field as a common key.

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook
* Visual Studio Code

## Learning Outcomes

This tutorial provided hands-on experience with the complete analytics workflow:

1. Loading raw data
2. Cleaning and preparing datasets
3. Joining multiple sources
4. Exploring relationships between variables
5. Interpreting analytical results
