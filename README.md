### Assignment 2 – OFKA Marketing Analytics

**Objective:** Determine whether website traffic behavior is linear or non-linear using machine learning models.

Topics Covered:

- Multiple Linear Regression (MLR)
- Random Forest Regression (RF)
- Feature Engineering
- Delta Analysis
- Model Evaluation
- Predictive Analytics

#### Results

|  Model  |    R²   |
|---------|---------|
| Multiple Linear Regression | 0.1619 |
| Random Forest Regression | 0.8709 |

The Random Forest model significantly outperformed the linear baseline, indicating strong non-linear relationships and contextual interactions within OFKA's website traffic data.
